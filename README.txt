Depenencias: 

EXPRESS
(Framework WEB)
https://expressjs.com

MONGOOSE
Trabajar con la base de datos

CONNECT-MONGO
Guardar secciones en base de datos

EXPRESS-SESSIONS
(Para trabajar con seccioens y cockies)

EXPRESS-HANDLEBARS
(template engine)

DOTENV
(para configuraciones y variables de entorno)

METHOD-OVERRIDE
(Por defecto solo se puede hacer GET POST, con esto podemos hacer PUT DELETE)

MOMENT
(para formatear fechas)

MORGAN
(Morgan sirve para hacer LOGs de las peticiones en la consola, solo lo ejecuto en desarrollo)

PASSPORT
(para autentificaciones, para que funcione bien tiene q estar instalado EXPRESS-SESSIONS
http://www.passportjs.org/packages/passport-google-oauth20/

PASSPORT-GOOGLE-OAUTH20
(para autentificaciones con google)

Resumen de dependencias:
npm i install express mongoose connect-mongo express-sessions express-handlebars dotenv method-override moment morgan passport passport-google-oauth20

Para iniciar:
npm i -D nodemon cross-env
